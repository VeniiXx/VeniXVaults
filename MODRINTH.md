# VeniXVaults

**VeniXVaults** is a fast, clean and advanced personal vault plugin for Paper 1.21.x - 1.26.x servers.

## Why VeniXVaults?

VeniXVaults gives players private storage through a polished GUI while giving server owners full control over limits, permissions, worlds, blacklisted items and backups.

## Highlights

- Personal vault GUI with `/depo`
- Direct vault opening with `/depo <number>`
- Permission-based vault count: `venixvaults.vaults.5`
- Permission-based vault size: `venixvaults.size.6`
- Admin vault inspection: `/venixvaults open <player> <vault>`
- Auto-save and save-on-close
- Built-in data backup command
- Item blacklist protection
- Disabled-world protection
- Hex-colored Turkish default messages
- No external database required

## Requirements

- Paper 1.21.x - 1.26.x
- Java 21+

## Commands

- `/depo`
- `/depo <number>`
- `/venixvaults help`
- `/venixvaults reload`
- `/venixvaults save`
- `/venixvaults backup`
- `/venixvaults open <player> <number>`
- `/venixvaults info <player>`

## Notes

VeniXVaults is an original plugin codebase. It is not a renamed or repackaged version of another vault plugin.
