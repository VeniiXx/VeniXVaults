# Changelog

## 1.0.0

- Initial public release for Modrinth.
- Supports Paper 1.21.x - 1.26.x API-compatible servers.
- Added personal vault selector GUI with `/depo`.
- Added direct vault opening with `/depo <number>`.
- Added default 1-vault setup with VIP vault permissions from 2 to 6.
- Added Turkish permission aliases such as `venixvaults.depos.6`.
- Added admin commands, backup command, autosave, instant save, blacklist, disabled worlds, and language files.
